KPIP_filter
===============
Filter UR StORFs

UR input generated by `-filt none` argument for StORF_finder.py

E.g.:

`python3 StORF_Finder.py -seq Genomes/E-coli/E-coli_URs.fasta -o ../testout/E-coli_output_no_filt -filt none`
